git

  • git checkout (ID) ->to check previous version of code
  • git checkot main ->to current version
  • git branch ->to show the current branch and all branch
  • git branch (branch name) -> to create a branch
  • git checkout (branch name) -> to switch branch
  • git branch --delete (branch name) -> to delete branch
  • git log & git log --all --oneline -> to show history
  • git log --all --oneline --graph -> to show graph history
